# Estudo de Caso Completo — Mercado Boa Vista

## 1. Contexto

O Mercado Boa Vista é um cenário de análise de dados voltado para demonstrar como um pequeno ou médio negócio pode transformar dados de vendas em decisões mais claras.

Este estudo de caso foi criado para apresentar um dashboard em Power BI com foco em indicadores comerciais, desempenho de produtos e recomendações práticas para melhoria dos resultados.

---

## 2. Desafio

O principal desafio analisado foi a dificuldade de transformar dados brutos de vendas em informações úteis para tomada de decisão.

Sem uma visualização organizada, o gestor pode ter dificuldade para identificar quais produtos vendem melhor, quais apresentam baixo desempenho e quais ações devem ser priorizadas.

---

## 3. Solução Desenvolvida

A solução foi a criação de um dashboard em Power BI com indicadores visuais, gráficos e elementos de storytelling com dados.

O dashboard foi pensado para permitir que o usuário entenda rapidamente a situação do negócio, identifique problemas e visualize possíveis ações de melhoria.

---

## 4. Indicadores Analisados

- Total de vendas;
- Quantidade de vendas;
- Desempenho por produto;
- Produtos com maior relevância comercial;
- Produtos com baixo desempenho;
- Recomendações de ações baseadas nos dados.

---

## 5. Ferramentas Utilizadas

- Power BI;
- Power Query;
- DAX;
- Git;
- GitHub.

---

## 6. Resultado Esperado

O resultado esperado é entregar uma apresentação visual clara, profissional e orientada à decisão, permitindo que o cliente compreenda o cenário do negócio e saiba quais ações tomar.
