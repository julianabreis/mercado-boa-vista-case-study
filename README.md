# Estudo de Caso — Mercado Boa Vista

## 1. Visão Geral

Este repositório apresenta um estudo de caso de análise de dados desenvolvido para o **Mercado Boa Vista**, com foco em transformar dados comerciais em informações visuais e acionáveis por meio de um dashboard em Power BI.

O objetivo principal do projeto é apoiar a tomada de decisão gerencial, permitindo uma visão clara sobre indicadores de vendas, produtos, desempenho comercial e oportunidades de melhoria.

---

## 2. Problema de Negócio

O Mercado Boa Vista precisava entender melhor seus dados de vendas para responder perguntas como:

- Qual é o desempenho geral das vendas?
- Quais produtos apresentam maior ou menor resultado?
- Quais indicadores ajudam a identificar problemas comerciais?
- Que ações podem ser tomadas com base nos dados?
- Como apresentar essas informações de forma clara para tomada de decisão?

---

## 3. Objetivo do Dashboard

O dashboard foi desenvolvido para apresentar, de forma visual e objetiva:

- Indicadores principais de desempenho;
- Análise de vendas;
- Análise por produto;
- Identificação de problemas e oportunidades;
- Recomendações de ações para melhoria dos resultados.

---

## 4. Ferramentas Utilizadas

- Power BI
- Power Query
- DAX
- Git
- GitHub

---

## 5. Estrutura do Repositório

- assets/images: imagens e capturas de tela do dashboard;
- assets/powerbi: arquivos do Power BI;
- data: bases de dados utilizadas ou dados fictícios/anonimizados;
- docs: documentação complementar do estudo de caso;
- exports: arquivos exportados, como PDF ou imagens finais.

---

## 6. Principais Análises

As análises do projeto buscam responder perguntas importantes para o negócio, como:

- Volume total de vendas;
- Quantidade de vendas realizadas;
- Produtos com melhor desempenho;
- Produtos com baixo desempenho;
- Possíveis gargalos comerciais;
- Ações recomendadas para melhorar os resultados.

---

## 7. Storytelling com Dados

O dashboard foi estruturado com foco em storytelling, para que o usuário consiga entender rapidamente:

1. O que está acontecendo;
2. Onde está o problema;
3. Qual indicador comprova o problema;
4. Que decisão pode ser tomada;
5. Quais ações devem ser priorizadas.
